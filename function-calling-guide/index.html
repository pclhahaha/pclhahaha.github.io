五、小结

Function Calling 的核心不是”LLM 调用函数”，而是”LLM 说出它想调用什么”。执行权始终在系统端——这意味着你可以加入权限控制、人工审批、调用审计等安全机制。这是 Agent 安全性的重要保障。
