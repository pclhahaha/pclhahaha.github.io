#### 多路复用

I/O 多路复用允许单个线程同时监听多个 I/O 通道的就绪状态，是实现高并发网络服务的关键技术。Linux 内核提供了三种多路复用机制：

| 机制   | 特点                                                         |
| ------ | ------------------------------------------------------------ |
| select | 最早的多路复用实现，使用位图存储 fd，默认最大 1024 个，每次调用需将 fd_set 从用户态拷贝到内核态，O(n) 遍历 |
| poll   | 使用 pollfd 结构数组，无 fd 数量限制，但仍需 O(n) 遍历       |
| epoll  | Linux 2.6+ 引入，基于红黑树和就绪链表，通过回调机制实现 O(1) 就绪事件获取，支持 ET/LT 模式 |
